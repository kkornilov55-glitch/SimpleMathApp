# Вычисления параметров треугольника

Консольное приложение на C# для вычисления геометрических параметров треугольника по трём сторонам.

## Описание

Программа предлагает пользователю меню из четырёх пунктов:
1. Вычислить полупериметр треугольника.
2. Вычислить площадь треугольника по формуле Герона.
3. Вычислить радиус вписанной и описанной окружностей.
4. Выход из программы.

После выбора задачи пользователь вводит длины сторон `a`, `b`, `c`. Программа проверяет корректность ввода (числовые значения, неотрицательность) и существование треугольника (неравенство треугольника, стороны больше нуля). Затем выполняются соответствующие расчёты, и результат выводится на экран с двумя знаками после запятой.

Все вычисления выполняются последовательно: полупериметр передаётся в функцию вычисления площади, а площадь — в функцию радиусов. Это позволяет избежать дублирования кода.

## Требования

- [.NET SDK](https://dotnet.microsoft.com/download) (версия 5.0 или выше, рекомендуется .NET 6/7/8).
- C# 7.0. Проект запускается в Visual Studio 2019+. Версия языка - .NET 8.0

## Запуск

1. [Клонируйте репозиторий](https://github.com/kkornilov55-glitch/ConsoleTriangleApp.git) или скопируйте файл `Program.cs` в локальную папку.
2. Откройте терминал в этой папке.
3. Выполните команду для запуска:
   ```bash
   dotnet run
   ```

Альтернативно, можно скомпилировать проект в исполняемый файл:
```bash
dotnet build
```
и затем запустить `Lab1.exe` из папки `bin/Debug/netX.X/`.

## Использование

При запуске отображается меню:

```
МЕНЮ
1. Вычислить полупериметр треугольника со сторонами a, b, c.
2. Вычислить площадь треугольника по формуле Герона.
3. Вычислить радиус вписанной (r) и радиус описанной (R) окружностей.
4. Выход.

Введите номер задачи:
```

Введите число от 1 до 4. Если выбран пункт 4, программа завершается. Иначе запрашиваются длины сторон:

```
Введите a: _
Введите b: _
Введите c: _
```

После ввода программа проверяет данные. Если обнаружена ошибка (некорректный номер, нечисловое значение, отрицательная длина, невозможность построить треугольник), выводится сообщение и предлагается нажать любую клавишу для возврата в меню.

При успешном вводе выводятся результаты:

- Для пункта 1: `p = ...`
- Для пункта 2: `S = ...`
- Для пункта 3: `r = ...` и `R = ...`

После вывода результатов программа ожидает нажатия клавиши и возвращается в меню.
