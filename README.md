# Time & Age Calculator

**Time & Age Calculator** — это простое Windows Forms приложение для конвертации времени и вычисления возраста по дате рождения. Программа состоит из двух основных модулей:

- **Конвертация времени** — перевод секунд в формат `ЧЧ:ММ:СС` и обратно.
- **Расчёт возраста** — определение полного количества лет, месяцев и дней, прошедших с даты рождения до текущего дня.

## Функциональные возможности

### 1. Конвертация секунд в формат времени
- Ввод целого числа секунд (от 0 до 216000).
- Проверка корректности ввода (только числа, неотрицательные, не превышающие лимит).
- Вывод результата в формате `ЧЧ:ММ:СС` с ведущими нулями.
- При ошибке выводится соответствующее сообщение.

### 2. Конвертация времени в секунды
- Ввод часов, минут и секунд (целые неотрицательные числа).
- Проверка корректности ввода.
- Вывод общего количества секунд.
- Обработка ошибок ввода.

### 3. Расчёт возраста
- Ввод дня, месяца и года рождения.
- Проверка корректности даты (учёт високосных годов, количества дней в месяцах).
- Расчёт полных лет, месяцев и дней на текущую дату.
- Если сегодня день рождения — поздравительное сообщение.
- Сообщение об ошибке при некорректной дате или если человек ещё не родился.

## Требования к системе
- Операционная система Windows.
- Установленная среда выполнения .NET (версия не ниже .NET Framework 4.7.2 или .NET 6/8 с поддержкой Windows Forms).
- Visual Studio (для сборки из исходников) или готовый исполняемый файл.

## Установка и запуск

### Из исходного кода
1. Склонируйте репозиторий или скачайте архив с исходными файлами.
2. Откройте решение (`.sln`) в Visual Studio.
3. Выберите конфигурацию `Release` и соберите проект.
4. Запустите полученный `.exe` файл.

### Готовый билд
Скопируйте папку с исполняемым файлом на компьютер и запустите `WinFormsApp1.exe`.

## Использование

### Вкладка "Секунды → Время"
- Введите количество секунд в поле **"Секунды"**.
- Нажмите кнопку **"Convert"**.
- Результат появится в поле **"Время (ЧЧ:ММ:СС)"** или сообщение об ошибке — в поле **"Error"**.

### Вкладка "Время → Секунды"
- Заполните поля **"Hours"**, **"Minuts"** (минуты), **"Seconds"**.
- Нажмите **"Convert"**.
- Результат отобразится в поле **"Seconds (result)"** или ошибка — в **"Error"**.

### Вкладка "Возраст"
- Введите день, месяц и год рождения в соответствующие поля (можно использовать подсказки `D`, `M`, `Year` как placeholder'ы).
- Нажмите **"Calculate"**.
- Результат (или сообщение об ошибке) появится в нижнем поле. При ошибке текст выделяется красным цветом.

## Примеры

| Действие                          | Ввод            | Ожидаемый вывод           |
|-----------------------------------|-----------------|---------------------------|
| Секунды → время                   | `3661`          | `01:01:01`                |
| Время → секунды                   | `01:01:01`      | `3661`                    |
| Возраст (сегодня 14.02.2026)      | `15.05.1990`    | `35 лет, 8 месяцев, 30 дней` (зависит от текущей даты) |
| День рождения сегодня              | `14.02.2000`    | `С днем рождения!!!!! Вам сегодня 26!` |

## Структура проекта

- **Class1.cs** — содержит два класса:
  - `Time` — методы `SecToTime` (секунды → время) и `TimeToSec` (время → секунды).
  - `Age` — метод `CalcAge` для расчёта возраста, а также вспомогательные методы `LeapYear` (проверка високосного года) и `DaysInSomeMonths` (количество дней в указанных месяцах года).
- **Form1.cs** — главная форма приложения, обработчики событий кнопок, проверка ввода, отображение результатов и ошибок.

## Примечания
- При конвертации времени максимальное значение секунд ограничено 216000 (60 часов).
- Расчёт возраста учитывает високосные годы, однако логика определения количества дней в месяцах использует упрощённое чередование (чётный месяц — 30 дней, нечётный — 31), что корректно для всех месяцев, кроме февраля и периода после июля. В текущей реализации это компенсируется отдельной обработкой февраля, но для месяцев август–декабрь может давать ошибку (например, август — чётный, но в нём 31 день). Рекомендуется использовать встроенные методы .NET для работы с датами.
